# Northwind Traders UI

Vue 3, Quasar, TypeScript frontend for the Northwind Traders Order Management System.

## Setup

```bash
npm install
npm run dev
```

The Quasar dev server defaults to port `9000`.

The frontend expects the API at `https://localhost:7001` during local development. Update `API_BASE_URL` in `quasar.config.ts` if the backend runs on a different URL.
